# GitHub Random Repository

Um buscador aleatório de repositórios do GitHub por linguagem de programação.

## 🚀 Como rodar
1. Clone o repositório.
2. Abra o `index.html` no seu navegador ou use um servidor local:
   ```bash
   python3 -m http.server 8000
   ```

## 🛠️ Tecnologias
- HTML5 / CSS3
- JavaScript Vanilla
- GitHub Search API
